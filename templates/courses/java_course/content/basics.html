Basics & Data Types

Introduction: Java offers a rich set of data types...

Primitive Data Types:

  • byte (8-bit) - Range: -128 to 127
  • short (16-bit) - Range: -32,768 to 32,767
  • int (32-bit) - Range: -2,147,483,648 to 2,147,483,647
  • long (64-bit) - Range: very large integers
  • float (32-bit) - Decimal numbers, e.g., 3.14f
  • double (64-bit) - More precise decimal numbers
  • char (16-bit) - Single character, e.g., 'A'
  • boolean - true or false

Variables Example:

int myNum = 5;              // Integer (whole number)
float myFloat = 5.99f;      // Floating point number
char myLetter = 'D';        // Character
boolean isJavaFun = true;   // Boolean

Type Casting: Converting between types...

int myInt = 9;
double myDouble = myInt;  // Automatic casting: int to double

// Manual casting
double myDouble2 = 9.78;
int myInt2 = (int) myDouble2;  // Narrowing conversion

Strings: A non-primitive but vital data type to handle text.

String greeting = "Hello, World!";
System.out.println(greeting);

Additional Tip: Always choose the data type that best fits the value range and memory efficiency...
