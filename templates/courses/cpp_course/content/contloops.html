Control Flow & Loops

Introduction: Control flow statements determine the order in which code executes. Loops allow repeating actions multiple times.

Conditional Statements

1. If-Else Example:

int age = 20;
if (age >= 18) {
    cout << "You are an adult." << endl;
} else {
    cout << "You are a minor." << endl;
}

2. Else If Ladder:

int marks = 85;
if (marks >= 90) {
    cout << "Grade A" << endl;
} else if (marks >= 75) {
    cout << "Grade B" << endl;
} else {
    cout << "Grade C" << endl;
}

3. Switch-Case Example:

int day = 3;
switch (day) {
    case 1: cout << "Monday"; break;
    case 2: cout << "Tuesday"; break;
    case 3: cout << "Wednesday"; break;
    default: cout << "Another day";
}

Loops

4. For Loop (Print 1 to 5):

for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}

5. While Loop (Countdown):

int count = 5;
while (count > 0) {
    cout << count << " ";
    count--;
}

6. Do-While Loop:

int num = 1;
do {
    cout << num << " ";
    num++;
} while (num <= 3);

Break & Continue

7. Using Break:

for (int i = 1; i <= 10; i++) {
    if (i == 5) break;
    cout << i << " ";
}

8. Using Continue:

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    cout << i << " ";
}

Combined Example: Sum of Even Numbers (1 to 10)

9. Full Example with If & Loop:

int sum = 0;
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        sum += i;
    }
}
cout << "Sum of even numbers: " << sum;

Additional Tip

Use loops to reduce repetitive code. Use break to exit early and continue to skip the current iteration!
