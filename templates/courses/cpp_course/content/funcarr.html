Functions & Arrays

Introduction: In C++, functions help organize code into reusable blocks, while arrays store multiple values of the same data type efficiently.

Functions in C++

1. Basic Function:

void greet() {
    cout << "Hello from function!" << endl;
}

int main() {
    greet();
    return 0;
}

2. Function with Parameters:

void add(int a, int b) {
    cout << "Sum: " << a + b << endl;
}

int main() {
    add(5, 3);
    return 0;
}

3. Function Returning a Value:

int square(int num) {
    return num * num;
}

int main() {
    cout << "Square: " << square(4);
    return 0;
}

4. Default Arguments:

void welcome(string name = "User") {
    cout << "Welcome, " << name << "!" << endl;
}

int main() {
    welcome();
    welcome("Alice");
}

5. Inline Functions: Used for small, frequently called functions.

inline int cube(int n) {
    return n * n * n;
}

int main() {
    cout << cube(3);
}

Arrays in C++

6. Declaring & Accessing Arrays:

int numbers[5] = {10, 20, 30, 40, 50};
cout << numbers[0];  // Prints 10

7. Looping Through Arrays:

int numbers[3] = {1, 2, 3};
for (int i = 0; i < 3; i++) {
    cout << numbers[i] << " ";
}

8. Taking Array Input:

int nums[3];
for (int i = 0; i < 3; i++) {
    cin >> nums[i];
}

Passing Arrays to Functions

9. Function to Print Array Elements:

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
}

int main() {
    int nums[4] = {5, 10, 15, 20};
    printArray(nums, 4);
}

2D Arrays

10. Declaring & Accessing 2D Arrays:

int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

cout << matrix[1][2];  // Prints 6

11. Looping Through 2D Array:

for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        cout << matrix[i][j] << " ";
    }
    cout << endl;
}

Extra Practice Examples

12. Find Max in Array:

int findMax(int arr[], int size) {
    int max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] > max)
            max = arr[i];
    }
    return max;
}

int main() {
    int nums[5] = {2, 8, 1, 9, 4};
    cout << "Max: " << findMax(nums, 5);
}

13. Sum of Array Elements:

int sumArray(int arr[], int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return sum;
}

Tip:

Functions make code modular and reusable. Arrays efficiently store and process data in bulk. Combined, they are powerful tools in C++ development!
