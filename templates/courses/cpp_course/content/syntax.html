Basics & Syntax

Introduction: C++ is a powerful language with strong system-level capabilities. It uses a structured syntax derived from C, but adds object-oriented features.

Basic Structure of C++ Program:

#include <iostream>
using namespace std;

int main() {
  cout << "Hello, World!";
  return 0;
}

Comments in C++: Use comments to explain code:

// This is a single-line comment

/* This is
   a multi-line comment */

Input and Output: Using cin and cout:

#include <iostream>
using namespace std;

int main() {
  int age;
  cout << "Enter your age: ";
  cin >> age;
  cout << "You are " << age << " years old.";
  return 0;
}

Variables and Data Types: Declare variables like this:

int myNum = 25;
float myFloat = 3.14;
char myChar = 'A';
bool isCppFun = true;
string greeting = "Hello";

If-Else Condition Example:

int number = 10;
if (number > 0) {
  cout << "Positive number";
} else {
  cout << "Non-positive number";
}

Loops Example (for loop):

for (int i = 1; i <= 5; i++) {
  cout << i << " ";
}

Function Example:

void greet() {
  cout << "Welcome to C++!";
}

int main() {
  greet();
  return 0;
}
