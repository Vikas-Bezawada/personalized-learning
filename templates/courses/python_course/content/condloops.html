Conditionals & Loops

In Python, conditionals and loops are essential building blocks of logic and flow control in a program. Conditionals allow you to execute code based on certain conditions, while loops allow you to repeat code multiple times. Understanding how to use both will help you create more powerful and dynamic programs.

üîé Conditionals: Making Decisions

Conditionals allow the program to execute different blocks of code based on certain conditions. In Python, we use if, elif, and else statements to control the flow of the program.

1. The if Statement

The if statement is used to test a condition. If the condition evaluates to True, the code inside the if block will execute. If it's False, the program will skip over the block.

Syntax: 

if condition:
    # code to execute if condition is True

Example:

age = 18
if age >= 18:
    print("You are an adult.")

In this example, the condition age >= 18 is True, so the message "You are an adult." will be printed.

2. The elif Statement

The elif (else if) statement allows you to test additional conditions if the initial if condition is False. You can have multiple elif statements in a chain.

Syntax: 

if condition1:
    # code to execute if condition1 is True
elif condition2:
    # code to execute if condition1 is False and condition2 is True
else:
    # code to execute if all conditions are False

Example:

age = 16
if age >= 18:
    print("You are an adult.")
elif age >= 13:
    print("You are a teenager.")
else:
    print("You are a child.")

In this example, since age = 16, the elif condition is True, so "You are a teenager." will be printed.

3. The else Statement

The else statement defines a block of code that will execute if all preceding conditions in the if and elif statements are False.

Example:

age = 10
if age >= 18:
    print("You are an adult.")
elif age >= 13:
    print("You are a teenager.")
else:
    print("You are a child.")

In this example, since age = 10, neither the if nor the elif conditions are true, so the code inside the else block runs, printing "You are a child."

4. Combining Conditions

You can combine multiple conditions using and, or, and not to form complex logical expressions.

  • and: Both conditions must be True for the entire expression to be True.
  • or: At least one condition must be True for the entire expression to be True.
  • not: Reverses the result of the condition.

Example:

age = 20
has_ticket = True
if age >= 18 and has_ticket:
    print("You can enter the event.")
else:
    print("You cannot enter the event.")

In this example, since both conditions are True, the message "You can enter the event." will be printed.

üîÑ Loops: Repeating Code

Loops allow you to execute a block of code multiple times, either for a fixed number of times or until a certain condition is met. Python supports two main types of loops: for loops and while loops.

1. The for Loop

The for loop is used to iterate over a sequence (like a list, string, tuple, or range) and execute a block of code for each item in the sequence.

Syntax: 

for item in sequence:
    # code to execute for each item

Example:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

This will print each fruit in the list:

  • apple
  • banana
  • cherry

2. The range() Function

The range() function generates a sequence of numbers, which is commonly used with for loops when you want to iterate over a specific range of numbers.

Example:

for i in range(5):
    print(i)

This will print the numbers from 0 to 4:

  • 0
  • 1
  • 2
  • 3
  • 4

3. The while Loop

The while loop runs as long as the specified condition is True. It‚Äôs commonly used when you don‚Äôt know beforehand how many times the loop will run.

Syntax: 

while condition:
    # code to execute while the condition is True

Example:

count = 0
while count < 5:
    print(count)
    count += 1  # Increase count by 1

This will print:

  • 0
  • 1
  • 2
  • 3
  • 4

4. Breaking Out of Loops

The break statement is used to exit a loop prematurely when a condition is met. The continue statement skips the current iteration and moves to the next one.

Example with break:

for i in range(10):
    if i == 5:
        break
    print(i)

This will print:

  • 0
  • 1
  • 2
  • 3
  • 4

Example with continue:

for i in range(5):
    if i == 2:
        continue
    print(i)

This will print:

  • 0
  • 1
  • 3
  • 4

üìù Nested Loops & Conditionals

You can nest loops and conditionals inside each other to create more complex logic. For example, you might want to loop over a list of lists or check multiple conditions within a loop.

Example of a nested loop:

for i in range(3):
    for j in range(2):
        print(f"i: {i}, j: {j}")

This will print:

  • i: 0, j: 0
  • i: 0, j: 1
  • i: 1, j: 0
  • i: 1, j: 1
  • i: 2, j: 0
  • i: 2, j: 1

Conclusion

Understanding conditionals and loops is crucial for controlling the flow of your Python programs. With conditionals, you can make decisions based on certain conditions, while loops allow you to repeat actions multiple times. Combined, they are powerful tools for creating dynamic, efficient, and logical programs.
