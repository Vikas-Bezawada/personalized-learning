Variables & Data Types

In Python, variables are used to store values, and the data type defines the type of value a variable can hold. Understanding how to properly use variables and their associated data types is crucial to writing Python programs effectively. This guide will help you understand the concept of variables, how to assign values, and how to use data types correctly.

üîé What are Variables?

A variable in Python is a symbolic name that refers to a value. You can think of a variable as a container where you can store data, and later retrieve or modify that data. Unlike some programming languages, Python doesn't require you to declare a variable type before assigning a value; it infers the type based on the value you assign to the variable.

Example: age = 25 assigns the value 25 to the variable age.

Variables can hold data of any type and can be reassigned to a new value, regardless of the type of the original value. Python is dynamically typed, meaning that the type of a variable is determined at runtime.

Variable Naming Rules

When naming variables, follow these rules:

  • Variable names must begin with a letter (a-z, A-Z) or an underscore (_).
  • Variable names can include letters, numbers (0-9), and underscores, but they cannot start with a number.
  • Variable names are case-sensitive (e.g., age is different from Age).
  • Avoid using Python keywords and built-in function names as variable names (e.g., class, def, print).

üìä Data Types in Python

Python supports several built-in data types, each serving a different purpose. Understanding these data types will allow you to work with various kinds of data effectively. Here‚Äôs an overview of the most commonly used data types in Python.

1. Numeric Types

  • int: Integer numbers. These are whole numbers (positive or negative) without decimal points. Examples: 1, -23, 0.
  • float: Floating-point numbers. These are real numbers that contain a decimal point. Examples: 3.14, -0.99, 2.0.

Example: 

age = 25  
height = 5.8

2. Text Type

  • str: A string is a sequence of characters enclosed in single, double, or triple quotes. Strings can hold text and also represent data like numbers, addresses, or even entire paragraphs of information. Examples: "Hello", 'Python', """Welcome to LearnSmart""".

Example: 

name = "Alice"  
message = 'Hello, World!'

3. Boolean Type

  • bool: A boolean type can only have two values: True or False. Booleans are typically used for conditional statements and logic control (e.g., if, while). Examples: True, False.

Example: 

is_active = True  
is_sunny = False

4. Sequence Types

  • list: A list is an ordered collection of items. Lists are mutable, meaning their content can be changed after creation. Lists can contain any type of data (e.g., numbers, strings, other lists). Examples: [1, 2, 3], ["apple", "banana"].
  • tuple: A tuple is similar to a list, but it is immutable, meaning once it is created, it cannot be modified. Examples: (1, 2, 3), ("a", "b", "c").

Example: 

numbers = [1, 2, 3, 4]  
coordinates = (4, 5)

5. Set Type

  • set: A set is an unordered collection of unique items. Sets are useful for eliminating duplicate values and performing mathematical operations like unions and intersections. Examples: {1, 2, 3}, {"apple", "banana"}.

Example: 

unique_numbers = {1, 2, 3}  
fruits = {"apple", "banana"}

6. Mapping Type

  • dict: A dictionary is an unordered collection of key-value pairs. The keys are unique and are used to retrieve the corresponding values. Examples: {"name": "Alice", "age": 25}.

Example: 

person = {"name": "Alice", "age": 25}

7. None Type

  • None: The special value None represents the absence of a value or a null value. It is commonly used for default values or when a variable is intentionally left uninitialized.

Example: 

result = None

üìù Assigning Values to Variables

Variables can be assigned values using the assignment operator =. In Python, you do not need to specify the type of data while declaring a variable. Python automatically identifies the type of data based on the value assigned to the variable.

Example:

x = 10  
name = "John"  
price = 15.75  
is_valid = True

‚öôÔ∏è Type Conversion

Sometimes, you need to convert between different data types. Python provides built-in functions to convert values from one data type to another.

Common Type Conversion Functions:

  • int(): Converts a value to an integer (if possible). E.g., int("10") converts the string "10" to the integer 10.
  • float(): Converts a value to a float. E.g., float("3.14") converts the string "3.14" to the float 3.14.
  • str(): Converts a value to a string. E.g., str(10) converts the integer 10 to the string "10".

Example of Type Conversion: 

age = "30"
age_int = int(age)  
height = 5.8
height_str = str(height)

‚ö° Common Operations on Variables

Python allows various operations to be performed on variables depending on their data type. Here are some common operations:

  • Arithmetic operations: + - * / % // ** for numeric data types.
  • String concatenation: Using + to combine strings: "Hello " + "World" results in "Hello World".
  • Membership operations: in checks if a value is present in a collection (list, tuple, set, or dictionary): 2 in [1, 2, 3] returns True.
  • Comparison operations: == != < > <= >= for comparing variables.

üßê Common Questions

Q: Can a variable hold multiple data types?

A: Yes, a variable can hold any data type. However, it can only hold one data type at a time. Python allows you to reassign a variable to a different type during execution.

Q: How do I check the data type of a variable?

A: You can check the data type of a variable using the type() function. Example: type(age) returns <class 'int'>.

Q: What is the difference between mutable and immutable types?

A: Immutable types cannot be modified after they are created (e.g., strings, tuples), whereas mutable types can be modified (e.g., lists, dictionaries).
